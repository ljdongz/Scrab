# Scrab

빠른 화면 캡처를 위한 가벼운 macOS 메뉴 바 앱. 단축키 하나로 화면의 원하는 영역을 캡처하고, 플로팅 썸네일 위젯에서 관리하며, 드래그 앤 드롭 또는 저장할 수 있습니다.

## 기능

- **즉시 캡처** — 글로벌 단축키로 macOS 기본 화면 선택 실행
- **플로팅 썸네일 위젯** — 캡처된 이미지가 번호 배지와 함께 사이드바에 표시
- **미리보기 창** — 썸네일을 클릭하면 복사/저장/삭제가 가능한 원본 크기 뷰어 실행
- **드래그 앤 드롭** — 썸네일을 다른 앱(Slack, Finder 등)으로 직접 드래그
- **클립보드 연동** — 캡처 시 이미지와 파일 경로가 자동으로 클립보드에 복사
- **일괄 작업** — 모든 캡처를 한 번에 저장하거나 삭제

## 요구 사항

- macOS 14.0 (Sonoma) 이상

## 설치

```bash
brew install --cask ljdongz/tap/scrab
```


## 사용법

### 캡처

`⌘ Shift S`를 눌러 화면 캡처를 시작합니다. 원하는 영역을 드래그하면 캡처된 이미지가 클립보드에 자동 복사되고 플로팅 썸네일 위젯에 표시됩니다.

- **이미지로 붙여넣기** — Slack, 메모 등 이미지를 지원하는 앱에서 `⌘ V`
- **파일 경로로 붙여넣기** — 터미널에서 `⌘ V`로 임시 파일 경로 붙여넣기

캡처 단축키는 설정에서 변경할 수 있습니다.

### 썸네일 위젯

캡처된 이미지가 번호 배지와 함께 플로팅 사이드바에 나열됩니다.

- 썸네일을 **클릭**하면 미리보기 창 실행
- **호버**하면 저장 및 삭제 버튼 표시
- 썸네일을 다른 앱(Finder, Slack 등)으로 **드래그 앤 드롭**
- 하단 툴바에서 **전체 저장** 또는 **전체 삭제**

### 미리보기 창

썸네일을 클릭하면 원본 크기 뷰어가 열립니다.

- **복사** — 이미지를 클립보드에 복사
- **저장** — 설정된 디렉토리에 저장
- **삭제** — 캡처 제거

### 저장

썸네일 또는 미리보기 창에서 저장 버튼을 클릭합니다. 이미지는 PNG 형식으로 설정된 저장 경로(기본값: `~/Desktop`)에 저장됩니다. 저장 후 터미널에서 `⌘ V`를 누르면 임시 경로 대신 저장된 파일 경로가 붙여넣어집니다.

### 임시 파일 정리

캡처 이미지와 드래그 캐시 파일은 시간이 지남에 따라 누적됩니다. **설정 > Temporary Files**에서 현재 파일 수와 용량을 확인하고 **Clear** 버튼으로 삭제할 수 있습니다.

## 설정

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| Save Location | 캡처 저장 디렉토리 | `~/Desktop` |
| Capture Shortcut | 화면 캡처 글로벌 단축키 | `⌘ Shift S` |
| Thumbnail Order | 최신순 또는 오래된순 | 최신순 |
| Capture Sound | 캡처 시 소리 재생 | 켜짐 |
| Launch at Login | 로그인 시 Scrab 자동 실행 | 꺼짐 |
